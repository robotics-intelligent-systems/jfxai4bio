# Artificial Intelligence in Bioinformatics

FastR is a high-performance implementation of the R programming language, built on GraalVM. BioJava is an open-source project dedicated to providing a Java library for processing biological data. Jenetics is a Genetic Algorithm, Evolutionary Algorithm, Grammatical Evolution, Genetic Programming, and Multi-objective Optimization library. ADAM is a genomics analysis platform with specialized file formats built using Apache Avro, Apache Spark, and Apache Parquet. Sparkling Water provides H2O functionality inside Spark cluster. Enso Analytics is a workflow-based data preparation platform designed for teams. Persistent cluster-friendly scheduler for Java. Pulsar is a distributed pub-sub messaging platform. Agrona provides a library of data structures and utility methods. Datumbox is an open-source Machine Learning framework. H2O Wave is a software stack for building realtime, browser-based applications and dashboards
